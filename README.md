# README #

Welcome to Guayaba Master API Generator

## Installation

```sh
$ npm install -g guayaba-gen
```


## Generate CRUD GrapHQL

```sh
$ guayaba-gen build:nest-crud-graphql
```

## Generate CRUD RestAPI

```sh
$ guayaba-gen build:nest-crud-restapi
```
